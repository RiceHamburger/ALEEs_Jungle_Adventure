# ALEEs Jungle Adventure

「ALEE's Jungle Adventure」は2Dアクションシューティングゲームです。
母星が破壊された猫人たちが生きられる惑星を探すため宇宙に行って、途中に間違って宇宙の生物の巣に足を踏み入れてしまった。<br>
プレイヤーは仲間を救うために未知のエリアに足を踏み入れる隊長'ALEE'として宇宙を冒険します。<br><br>
